## 🔧 구현된 기능

### 1️⃣ AdventureScreen (모험 전투)
```javascript
✅ 전투 시작: [0, 50, 50, 50, 50, 50]
✅ 공격: [0, 100, 50, 100]
✅ 피격: [0, 200, 100, 200]
✅ 승리: [0, 100, 50, 100, 50, 100]
✅ 패배: 500ms 진동
```

**트리거:**
- 스테이지 선택 → 전투 시작 효과음
- 스킬 사용 → 공격 효과음
- 적 피해 → 피격 효과음
- 전투 승리 → 승리 효과음

### 2️⃣ CompendiumScreen (몬스터 도감)
```javascript
✅ Rare: [0, 100]
✅ Epic: [0, 50, 50, 100]
✅ Unique: [0, 100, 50, 50, 100]
✅ Legendary: [0, 200, 100, 200]
```

**트리거:**
- 몬스터 클릭 → 레어도별 효과음

### 3️⃣ DungeonScreen (퀴즈 던전)
```javascript
✅ 정답: [0, 100, 50, 100]
✅ 오답: [0, 200]
✅ 완료: [0, 100, 50, 100, 50, 100, 50, 100]
```

**트리거:**
- 정답 선택 → 정답 효과음
- 오답 선택 → 오답 효과음
- 던전 완료 → 완료 효과음

### 4️⃣ ShopScreen (상점)
```javascript
✅ 구매: [0, 100, 50, 100, 50, 100]
✅ 오류: 200ms 진동
```

**트리거:**
- 아이템 구매 → 구매 효과음
- 돈 부족 → 오류 효과음

## 📱 사운드 시스템 특징

### Vibration API 활용
- **플랫폼 호환성**: Android & iOS 완벽 지원
- **배터리 효율**: 진동으로 배터리 소비 최소화
- **리소스**: 별도 오디오 파일 필요 없음
- **즉시 반응**: 네트워크 지연 없음

### 진동 패턴 설계
- **짧은 진동** (50-100ms): 빠른 액션
- **긴 진동** (200ms): 중요한 이벤트
- **연속 진동**: 특수 효과

## 🎯 사용자 경험

### 시각적 + 촉각적 피드백
| 액션 | 시각 | 촉각 | 효과 |
|------|------|------|------|
| 전투 시작 | UI 활성화 | 다중 진동 | 긴장감 조성 |
| 스킬 사용 | 데미지 표시 | 빠른 진동 | 즉시성 강화 |
| 적 피해 | HP 감소 | 강한 진동 | 영향력 표현 |
| 승리 | 화면 강조 | 성공 패턴 | 쾌감 증대 |
| 몬스터 클릭 | 모달 열림 | 레어도별 패턴 | 차별성 표현 |

## 🔄 비교: 웹 vs 모바일

| 요소 | 웹 앱 | 모바일 앱 |
|------|------|--------|
| **기술** | Web Audio API | Vibration API |
| **사운드** | 실제 오디오 | 촉각 피드백 |
| **배경음** | O | - |
| **효과음** | O | O |
| **플랫폼** | 브라우저 | Android/iOS |
| **리소스** | 오디오 파일 | 내장 API |
| **배터리** | 중간 | 낮음 |
