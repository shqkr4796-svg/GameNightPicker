## ⚠️ **빠진 항목 (웹에 있지만 모바일에 없음)**

### 🔴 **필수 기능** (우선순위: 높음)

#### 1️⃣ Dashboard (대시보드) - **⭐ 가장 중요**
```
웹 기능:
- 플레이어 상태 (레벨, 경험치, 날짜, 시간, 건강, 기력, 돈, 스탯포인트)
- 능력치 정보 (힘, 지능, 외모, 체력, 운)
- 티어 시스템 (랭크 진행 상황)
- 현재상황 (직업, 부동산, 스킬 정보)
- 능력치 차트 (그래프)
- 최근 이벤트

모바일 상태: ❌ 없음
우선순위: 🔴 높음 (게임의 핵심 정보)
```

#### 2️⃣ Inventory (인벤토리)
```
웹 기능:
- 무기 목록
- 방어구 목록
- 스킬 아이템 (스킬 리차저, 리셋터 등)
- 장착/해제 기능
- 아이템 상세 정보

모바일 상태: ❌ 없음
우선순위: 🔴 높음 (게임 진행에 필수)
```

#### 3️⃣ Achievements (성취 목록)
```
웹 기능:
- 성취 목록 (20+ 성취)
- 난이도별 분류 (쉬움, 보통, 어려움, 극어려움)
- 성취 포인트 (총점 표시)
- 진행 상황 세부 정보

모바일 상태: ❌ 없음
우선순위: 🟡 중간 (보상/동기부여 시스템)
```

#### 4️⃣ Quiz/Word Management (단어 퀴즈)
```
웹 기능:
- 카테고리별 단어 퀴즈
- 단어 추가/삭제/편집
- 틀린 문제 재도전
- 진행률 추적

모바일 상태: ❌ 없음
우선순위: 🟡 중간 (학습 시스템)
```

### 🟡 **선택적 기능** (우선순위: 낮음)

#### 5️⃣ Allocate Stats (스탯 할당)
```
웹 기능:
- 스탯 포인트 사용해서 능력치 증가
- 실시간 스탯 변경

모바일 상태: ❌ 없음
우선순위: 🟢 낮음 (SkillsScreen에서 통합 가능)
```

#### 6️⃣ Sleep/Rest (휴식)
```
웹 기능:
- 게임 내 휴식
- 건강/기력 회복
- 레벨업 보너스

모바일 상태: ❌ 없음 (일부 기능만 있을 수 있음)
우선순위: 🟢 낮음 (MainHubScreen에서 통합 가능)
```

#### 7️⃣ All Monsters (전체 몬스터 목록)
```
웹 기능:
- 모든 몬스터 데이터베이스 조회
- 상세 정보 (스탯, 스킬, 드롭 아이템)

모바일 상태: ❌ 없음 (CompendiumScreen에 포함됨)
우선순위: 🟢 낮음 (CompendiumScreen과 중복)
```

